# TungBox TODO

最后更新：2026-06-02 | 当前版本 v0.1.1(0051)

TungBox 是面向 macOS 的 sing-box 图形客户端，目标是为日常代理需求提供开箱即用的体验。

## 已完成 ✅

- ✅ 左侧一级菜单（首页/节点/规则/订阅/连接/日志/设置）+ MD3 风格主界面
- ✅ 首页仪表盘：系统代理/TUN/出站模式切换，实时上行/下行、连接数，URLTest 延迟卡片
- ✅ 订阅功能：URL 添加 / 本地文件导入 / 剪贴板导入，编辑、删除、刷新
- ✅ 订阅格式：自动识别 sing-box JSON + Clash YAML，非标准格式可理解错误提示
- ✅ 订阅自动刷新可配置（关闭 / 30 分钟 / 1-24 小时），刷新失败系统通知 + 卡片错误状态
- ✅ 出站模式：规则/全局/直连三模式 + 基础分流规则 + 规则集 SRS 下载 & 展开
- ✅ 自定义规则：按订阅独立存储，添加 / 编辑 / 启用禁用 / 删除，刷新后自动合并
- ✅ 节点页：代理分组网格、节点选择、单节点/批量 URLTest 延迟测试、自动选择最快节点
- ✅ 运行控制：sing-box 启动/停止、配置保存/检查/格式化、普通/管理员两种启动模式
- ✅ Core 管理：检测、导入、从 GitHub 自动安装最新版、旧版测试、更新检查
- ✅ TUN 服务：LaunchDaemon 安装/卸载/启用/禁用，启动时崩溃恢复
- ✅ 状态栏：系统代理、TUN、出站模式、代理组快速切换、显示控制台、退出
- ✅ 后台运行：关闭窗口隐藏 Dock，保留状态栏，可从状态栏恢复
- ✅ 开机自启动 & 静默启动（设置页开关）
- ✅ 连接页：实时列表 + 按节点/域名/IP/规则过滤 + 右键关闭单条/全部连接
- ✅ 日志页：按级别过滤（INFO/WARN/ERROR/DEBUG）+ 关键词搜索 + 行数统计 + 复制导出
- ✅ 发布打包：自动编译瘦身版 Core → .app 签名 → .dmg（~15 MB）
- ✅ 连接实时流量（通过 Clash API 拉取上行/下行字节增量）
- ✅ P0 全项：首次启动零状态 / 退出清理链路 / TUN 全链路 / Core 更新 / 应用图标

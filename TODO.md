# TungBox TODO

最后更新：2026-06-02 | 当前版本 v0.1.1(0038)

TungBox 是面向 macOS 的 sing-box 图形客户端。参考竞品：Clash Verge Rev（开源/社区驱动）、Surge（商业/iOS-first）、ShadowsocksX-NG（轻量单协议）。

## P1 — 竞品对齐（Clash Verge Rev 参照）

Clash Verge Rev 是当前 macOS 开源代理客户端标杆，以下为其核心功能中 TungBox 尚缺或未完整的。

### 订阅 & 配置

- [ ] **Clash / 通用格式导入**：识别 Clash YAML、vmess/vless/ss/trojan share-link、非 sing-box JSON 时给出可理解错误提示（当前仅解析 sing-box JSON + Base64）。
- [ ] **订阅自动刷新可配置**：设置页面暴露刷新间隔（当前固定 Hourly，无法调整）。
- [ ] **导入方式多样化**：支持 URL 导入、本地文件导入。

### 节点 & 策略

- [ ] **多协议节点覆盖验证**：需用真实机场验证 vmess / vless / trojan / shadowsocks / hysteria2 / tuic / wireguard 全部协议可入站、可测速、可切换。
- [ ] **URLTest 切换后断开旧连接**：确保自动选择切到新节点时主动断开旧代理连接，出口 IP 立即变化。
- [ ] **节点切换一致性**：UI 列表 ↔ 状态栏 Menu ↔ sing-box API ↔ 配置文件 四个状态源对外呈现一致。
- [ ] **策略组增强**：支持 Failover（fallback 链路）、Load Balance 策略组（当前仅 URLTest + Selector）。

### 连接页

- [ ] **连接过滤 & 搜索**：按节点/Domain/IP/规则过滤连接列表（当前仅全部展示）。
- [ ] **单条连接关闭**：右键或按钮关闭单条连接（当前仅有"关闭全部"）。
- [ ] **连接实时速率**：单条连接展示实时上行/下行速率（当前仅展示主机的总速率）。

### 规则

- [ ] **自定义规则启用/禁用/编辑/拖拽排序**：当前仅支持添加 & 删除，缺少开关和编辑入口。
- [ ] **规则集缓存管理**：展示 SRS 规则集下载状态 + 失败重试 + 手动刷新 + 清理缓存。

### 日志

- [ ] **日志分级过滤 + 搜索**：支持按 INFO/WARN/ERROR/DEBUG 过滤 + 关键词搜索（当前仅追加 + 清空）。
- [ ] **日志导出**：支持复制。

## P2 — 差异化 & 体验

参考 Surge 和 Clash Verge Rev 中提升日常使用感受但非阻塞的功能。

- [ ] **流量统计持久化**：今日 / 近 7 天 / 近 30 天流量柱状图在重启后仍有数据（当前仅在内存中累积）。
- [ ] **代理冲突检测**：启动时检测 Surge / Clash / 其他代理端口占用并弹窗提示关闭。
- [ ] **窗口状态持久化**：记住窗口大小、位置、最后打开 Tab、深浅色模式（当前实现了颜色切换但未持久化位置）。
- [ ] **增强模式（TUN fallback）**：无管理员权限时 TUN 无法启动给出明确指引，且在常规 HTTP/SOCKS 代理模式下也能完整工作。

## P3 — 工程化 & 测试

- [ ] **最小 UI 冒烟**：构建 → 启动 → 切 6 个 Tab → 添加订阅 → 刷新 → 切换节点 → 启动代理 → 关闭 → 不崩溃。
- [ ] **核心逻辑单测**：订阅解析（多格式 fault injection）、自定义规则合并（冲突/覆盖/排序）、规则生成 JSON schema、版本号比较（Core 更新）。
